# CSIE_ADMS_Back-End

## Setup
1. `cp ./manifest/cofig.yaml.example ./manifest/config.yaml`
2. 編輯 `./manifest/config.yaml`，修改相關資訊
3. 執行 `gf run main.go`

## 預設帳號
- Email：`admin@csie.adms`
- Password：`1234567890`

> [!IMPORTANT]
> 實際使用時，登入後請記得修改帳號、密碼
